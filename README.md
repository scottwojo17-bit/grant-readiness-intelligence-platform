# Grant Readiness Intelligence Platform - Demo Version

This repository is the buyer-facing evaluation demo. It is intentionally limited.

## What This Demo Does

- Accepts a grant opportunity URL or reference.
- Produces a basic readiness score.
- Shows simple strengths, gaps, and next actions.
- Lets a reviewer adjust basic readiness fields.

## What Is Not Included

- Production scoring model.
- Expanded analysis features.
- Saved records or account system.
- Workflow automation.
- Advanced review tools.
- Product roadmap or operating framework.

## Owner Note

Keep the full product source, original planning language, private methods, roadmap, and production logic in a separate private repository. This demo repository can be shared or deployed for evaluation without exposing the full build.
